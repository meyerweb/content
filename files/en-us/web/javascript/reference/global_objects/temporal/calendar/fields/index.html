The fields() method TK. This method does not need to be called directly except in specialized code. It is called indirectly when using the from() and with() methods of the {{jsxref('Temporal/PlainDateTime','Temporal.PlainDateTime')}}, {{jsxref('Temporal/PlainDate','Temporal.PlainDate')}}, {{jsxref('Temporal/PlainMonthDay','Temporal.PlainMonthDay')}}, {{jsxref('Temporal/PlainYearMonth','Temporal.PlainYearMonth')}}, and {{jsxref('Temporal/ZonedDateTime','Temporal.ZonedDateTime')}}, classes.

Syntax

fields(fieldList);

Parameters

fieldList

A list of field names represented as an array of strings or other iterable yielding strings. TK

Return value

An array with a list of field names as strings.

Examples

// In the ISO calendar, this method just makes a copy of the input array
Temporal.Calendar.from('iso8601').fields(['monthCode', 'day']);
// => [ 'monthCode', 'day' ]
