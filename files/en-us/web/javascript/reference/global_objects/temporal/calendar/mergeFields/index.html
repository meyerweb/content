The mergeFields() method TK. This method does not need to be called directly except in specialized code. It is called indirectly when using the from() and with() methods of the {{jsxref('Temporal/PlainDateTime','Temporal.PlainDateTime')}}, {{jsxref('Temporal/PlainDate','Temporal.PlainDate')}}, {{jsxref('Temporal/PlainMonthDay','Temporal.PlainMonthDay')}}, {{jsxref('Temporal/PlainYearMonth','Temporal.PlainYearMonth')}}, and {{jsxref('Temporal/ZonedDateTime','Temporal.ZonedDateTime')}}, classes.

Syntax

fields(fieldList, additionalFields);

Parameters

fieldList

A list of field names represented as an array of strings or other iterable yielding strings. TK

additionalFields

Another plain object with properties representing calendar units. TK

Return value

A new object with properties from both fieldList and additionalFields.

Examples

// In built-in calendars, this method copies properties, taking `month`
// and `monthCode` into account
Temporal.Calendar.from('iso8601').mergeFields(
  { year: 2006, month: 7, day: 31 },
  { monthCode: 'M08' }
);
// => { year: 2006, monthCode: 'M08', day: 31 }
