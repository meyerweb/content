The calendar read-only property returns a {{jsxref('Temporal/Calendar','Temporal.Calendar')}} object containing the ISO 8601 calendar. No other calendars are possible in Temporal 1.0, but the property exists for purposes of future compatibility.

Syntax

var timeCalendar = time.calendar;

Value

A {{jsxref('Temporal/Calendar','Temporal.Calendar')}} object containing the ISO 8601 calendar.
