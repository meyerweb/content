The getISOFields() method returns an object containing the ISO 8601 calendar fields and values that correspond to the month and day. This method is mainly useful if you are implementing a custom calendar.

Syntax

date.getISOFields();

Parameters

None.

Return value

An object containing the isoYear, isoMonth, isoDay, and calendar properties. The value of the isoYear property will be equal to the referenceISOYear constructor argument passed when the {{jsxref('Temporal.PlainMonthDay','Temporal.PlainMonthDay')}} object was {{jsxref('Temporal/PlainMonthDay/constructor','constructed')}}.

Examples

md = Temporal.PlainMonthDay.from('04-22');
md.getISOFields().isoDay;   // => 22
md.getISOFields().isoMonth; // => 4
md.getISOFields().isoYear;  // => 1972

md = new Temporal.PlainMonthDay(4, 22, 'iso8601', 2021);
md.getISOFields().isoDay;   // => 22
md.getISOFields().isoMonth; // => 4
md.getISOFields().isoYear;  // => 2021

md = new Temporal.PlainMonthDay(4, 22, 'hebrew', 5781);
md.getISOFields().isoDay;   // => 22
md.getISOFields().isoMonth; // => 4
md.getISOFields().isoYear;  // => 5781  TK TK TK
