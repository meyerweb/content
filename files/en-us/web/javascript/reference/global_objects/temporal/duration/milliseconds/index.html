The milliseconds read-only property returns the value of the milliseconds component of the {{jsxref('Temporal.Duration','Temporal.Duration')}} object.

Syntax

var milliseconds = duration.milliseconds;

Value

An integer.

Examples

d = Temporal.Duration.from('P1Y2M3W4DT5H6M7.987654321S');
d.milliseconds; // => 987
