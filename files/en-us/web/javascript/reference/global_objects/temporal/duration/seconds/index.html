The seconds read-only property returns the value of the seconds component of the {{jsxref('Temporal.Duration','Temporal.Duration')}} object.

Syntax

duration.seconds

Value

An integer.

Examples

d = Temporal.Duration.from('P1Y2M3W4DT5H6M7.987654321S');
d.seconds; // => 7
