The daysInWeek read-only property gives the number of days in the week containing the date.

Syntax

datetime.daysInWeek

Value

An integer representing the number of days in the week in which the date occurs. In the ISO 8601 calendar, this is always 7, but in other calendar systems the value may vary from week to week.

Examples

datetime = Temporal.PlainDateTime.from('2021-04-27');
datetime.daysInWeek; // => 7
