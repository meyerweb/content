The hour read-only property returns a number that identifies the hour component of the Temporal.ZonedDateTime object.The number is an integer in the range 0 through 23, inclusive.

Syntax

datetime.hour

Value

An integer representing the hour component.

Examples

dt = Temporal.ZonedDateTime.from('1995-12-07T03:24:30.000003500');
dt.hour; // => 3
