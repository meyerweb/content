The nanosecond read-only property returns a number that identifies the nanosecond component of the Temporal.ZonedDateTime object. The number is an integer in the range 0 through 999, inclusive.

Syntax

datetime.nanosecond

Value

An integer representing the nanosecond component.

Examples

dt = Temporal.ZonedDateTime.from('1995-12-07T03:24:30.000003500');
dt.nanosecond;  // => 500
