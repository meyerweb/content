A Temporal.PlainYearMonth represents a particular month on the calendar. For example, it could be used to represent a particular instance of a monthly recurring event, like "the June 2019 meeting".

Temporal.PlainYearMonth refers to the whole of a specific month; if you need to refer to a calendar event on a certain day, use {{jsxref('Temporal.PlainDate','Temporal.PlainDate')}} or even {{jsxref('Temporal.PlainDateTime','Temporal.PlainDateTime')}}. A Temporal.PlainYearMonth can be converted into a {{jsxref('Temporal.PlainDate','Temporal.PlainDate')}} by combining it with a day of the month, using the {{jsxref('Temporal.PlainYearMonth/toPlainDate','toPlainDate()')}} method.

Constructor

new {{jsxref('Temporal/PlainYearMonth/constructor','Temporal.PlainYearMonth()')}}

Creates a new Temporal.PlainYearMonth object which represents a combination of year and month, with no associated single day.
