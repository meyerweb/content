The instant() method returns an object representing the current system time. It works like the old-style JavaScript {{jsxref("date/now","Date.now()")}}, but with accuracy in nanoseconds instead of milliseconds.

Syntax

Temporal.now.instant()

There are no parameters.

Return value

A Temporal.Instant object representing the current system time, without regard to calendar or time zone, measured in nanoseconds. By default, the system time is measured as nanoseconds elapsed since January 1, 1970 00:00:00 UTC.

Examples

var timestamp = Temporal.now.instant();
function timeit(func) {
  start = Temporal.now.instant();
  try {
    return func();
  } finally {
    end = Temporal.now.instant();
    console.log(`The function took ${end.since(start)}`);
  }
}
timeit(() => JSON.parse(someData));
// Console output:
// The function took PT0.001031756S
