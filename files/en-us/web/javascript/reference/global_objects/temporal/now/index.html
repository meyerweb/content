The Temporal.now object has several methods which give information about the current time and date.

Methods

{{jsxref("Temporal/now/instant", "Temporal.now.instant()")}}
Returns the current system time in nanoseconds as a new Temporal.Instant object.
{{jsxref("Temporal/now/timeZone", "Temporal.now.timeZone()")}}
Returns the current system time zone as a Temporal.TimeZone object.
{{jsxref("Temporal/now/zonedDateTime", "Temporal.now.zonedDateTime(calendar)")}}
Returns the current system date, time, time zone, and time zone offset in the reckoning of a given calendar as a Temporal.ZonedDateTime object.
{{jsxref("Temporal/now/zonedDateTimeISO", "Temporal.now.zonedDateTimeISO()")}}
Returns the current system date, time, time zone, and time zone offset in the reckoning of the ISO 8601 calendar as a Temporal.ZonedDateTime object.
{{jsxref("Temporal/now/plainDate", "Temporal.now.plainDate(calendar)")}}
Returns the current system date in the reckoning of the given calendar as a Temporal.PlainDate object.
{{jsxref("Temporal/now/plainDateISO", "Temporal.now.plainDateISO()")}}
Returns the current system date in the reckoning of the ISO 8601 calendar as a Temporal.PlainDate object.
{{jsxref("Temporal/now/plainTimeISO", "Temporal.now.plainTimeISO()")}}
Returns the current system time in the reckoning of the ISO 8601 calendar as a Temporal.PlainTime object.
{{jsxref("Temporal/now/plainDateTime", "Temporal.now.plainDateTime(calendar)")}}
Returns the current system date and time in the reckoning of the given calendar as a Temporal.PlainDateTime object.
{{jsxref("Temporal/now/plainDateTimeISO", "Temporal.now.plainDateTimeISO()")}}
Returns the current system date and time in the reckoning of the ISO 8601 calendar as a Temporal.PlainDateTime object.
